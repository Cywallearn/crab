# Getting Started with Cywallearn CRAB v1

## Welcome!

Cywallearn CRAB v1 transforms your Android device into a Kali Linux-grade penetration testing machine.

## First Launch

Run `bash main.sh` to start Cywallearn.

You'll see:
1. The Cywallearn banner
2. System status check
3. Interactive shell prompt

## Navigation

- `menu` - Open the main menu
- `help` - Open the learning center
- `tools` - Browse all 50+ tools
- `terminal` - Launch the advanced terminal

## Your First Scan

1. Type `menu` to open the main menu
2. Select `[1] Reconnaissance`
3. Choose `Nmap`
4. Enter your target IP when prompted

## Pro Tips

- Use tab completion in the terminal
- All tools have `--help` flags
- Sessions are auto-logged
- Use `quick` for automated attacks
